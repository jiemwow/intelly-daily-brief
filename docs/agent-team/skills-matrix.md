# Skills Matrix

## 基础技能

| Skill | 用途 | 主要服务角色 |
| --- | --- | --- |
| `openai-docs` | 核验 OpenAI 官方文档与最新能力 | 资料验证 Agent |
| `imagegen` | 生成或探索视觉概念、素材变体 | 视觉体验 Agent |
| `agent-tools` | 多模型调用、外部搜索、图像/视频工具 | 产品研究 / 资料验证 / 视觉体验 |
| `skill-installer` | 后续补装新技能 | 主负责人 |

## 新增通用技能

| Skill | 用途 | 主要服务角色 |
| --- | --- | --- |
| `doc` | 输出更结构化的文档与规范 | 产品研究 / 工程架构 |
| `frontend-skill` | 高质量网页、原型、视觉结构与界面实现思路 | 视觉体验 / 工程架构 |
| `playwright` | 页面自动化验证、网页流程检查 | 资料验证 / 工程架构 |
| `screenshot` | 页面截图、参考采样、视觉比对 | 视觉体验 / 产品研究 |

## 暂不默认安装

以下类别暂不纳入默认包：

- Figma 相关技能
- Notion 相关技能
- 部署平台专用技能
- 安全专项技能

原因：

- 新项目前期以定义、研究、视觉和工程方案为主
- 这些技能往往依赖额外外部平台或更明确的执行场景
- 等项目进入具体实现阶段再按需补装更合理
